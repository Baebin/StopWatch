# StopWatch
**Android Studio**
##
**프로그램 설명**

간단한 시간 측정 어플리케이션입니다.

스탑워치로서 시간을 재면서
구간 기록을 통해 시간을 기록할 수 있습니다.

프로그램 이름: StopWatch

개발자: Baebin (baebine@naver.com)

기능:
1. 시간을 측정할 수 있습니다.
2. 시간을 기록할 수 있습니다.

언어: Java
